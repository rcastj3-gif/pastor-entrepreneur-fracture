## CUSTOMIZATION QUICK REFERENCE

### Change Primary Gold Color
```css
/* In styles.css, line ~10 */
--color-primary: #d4af37;  /* Change this */
```

### Adjust Section Spacing
```css
/* In styles.css, line ~35 */
--spacing-4xl: 6rem;  /* Change to 5rem or 7rem */
```

### Modify Button Hover
```css
.btn-primary:hover {
    background-color: var(--color-primary-dark);
    transform: translateY(-2px);  /* Change -2px to different value */
    box-shadow: var(--shadow-gold);
}
```

### Add New Font Size
```css
:root {
    --font-size-sm: 0.875rem;  /* Add new variable */
}
p.small { font-size: var(--font-size-sm); }
```
